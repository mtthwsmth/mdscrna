# mdscrna

<!-- badges: start -->

<!-- badges: end -->

The goal of mdscrna is to store convenience packages I use in my
scRNAseq analyses in a way that I can easily manage across analyses,
make available to collaborators, and to make it available for use in
published analyses.

## Installation

You can install the development version of mdscrna from
[GitHub](https://github.com/) with:

``` r
# install.packages("pak")
pak::pak("mtthwsmth/mdscrna")
#> ℹ Loading metadata database✔ Loading metadata database ... done
#>  
#> ℹ No downloads are needed
#> ✔ 1 pkg: kept 1 [3.5s]
```
